# Tuition Track Frontend

React, Vite, and Tailwind CSS frontend for the Tuition Track Management System.

## Setup

```sh
bun install
```

## Run

Start the backend first on `http://localhost:3000`, then run:

```sh
bun run dev
```

Frontend: `http://localhost:5173`

The Vite dev server proxies `/api` requests to the backend.

## Build and Lint

```sh
bun run build
bun run lint
```

Implemented screens include dashboard, students, teachers, classes/enrollment, attendance, and payments.
